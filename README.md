<h2>🌟 Overview</h2>

<p>
InterviewIQ is a full-stack SaaS platform that simulates real interview environments.
It helps users improve interview skills using AI-generated questions, resume analysis, and intelligent feedback system.
</p>

<hr/>

<h2>✨ Features</h2>

<ul>
  <li>📄 Upload Resume (PDF/Text Analysis)</li>
  <li>🤖 AI-Generated Interview Questions</li>
  <li>🎯 Technical, Behavioral & HR Mock Interviews</li>
  <li>📊 Instant Answer Scoring & Feedback</li>
  <li>🧠 Resume ATS Score Analyzer</li>
  <li>💳 Credit-Based Access System</li>
  <li>💰 Razorpay Payment Integration</li>
  <li>📈 Progress Tracking Dashboard</li>
  <li>⚡ Smooth UI with Framer Motion</li>
</ul>

<hr/>

<h2>🧠 How It Works</h2>

<pre>
Login → Upload Resume → Select Role → Start Interview
     → Answer Questions → AI Evaluation → Feedback → Score Tracking
</pre>
